# MathAffiliation

An online learning platform built with HTML, CSS, JavaScript, and Firebase.

## Live Demo
https://mathaffiliation.netlify.app

## Features
- Quiz system with scoring and review
- Admin dashboard
- Tutor and learner modules
- Authentication (login/signup/reset)

## Tech Stack
- HTML
- CSS
- JavaScript
- Firebase
